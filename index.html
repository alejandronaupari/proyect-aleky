El Esqueleto: HTML

HTML es la estructura de una página web. Define dónde van los textos, las imágenes y los botones. Es como los ladrillos y columnas de una casa.

🎨

El Diseño: CSS

CSS es el estilo y el arte. Se encarga de los colores oscuros elegantes, los bordes redondeados, las fuentes hermosas y los brillos morados que estás viendo aquí.

JavaScript le da vida a la página. Permite que las cosas reaccionen cuando haces clic en ellas. ¡Haz clic en este botón para ver la magia!
